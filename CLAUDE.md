## 業界ルール・慣習

詳細な用語集: [業界用語集.md](./業界用語集.md)

### 基本的な仕組み
- **三店方式**: パチンコの換金システム。「ホール → 景品交換所 → 景品買取業者」の3ヵ所を経由する
- **等価交換**: 玉1個=4円、コイン1枚=20円で、借りた時と交換時の価値が同じ
- **換金率**: 玉を換金する際の玉1個あたりの値段（等価交換以外のホールもある）
- **風営法（風適法）**: 「風俗営業等の規制及び業務の適正化等に関する法律」。パチンコホールの営業を規制する法律
- **検定**: 遊技機が保通協の型式試験に合格後、公安委員会に申請するもの。有効期間は3年
- **保通協**: 財団法人・保安電子通信技術協会。パチンコ・パチスロの新台検査機関

### ホール設備
- **島**: 台を設置するための設備。1列を一島と言う
- **サンド**: 台と台の間にある、玉/コインを貸し出す機械
- **CRユニット**: カードリーダー機。プリペイドカードを差し込んで玉を借りる
- **ジェットカウンター**: 獲得した出玉の個数を計測する機械
- **ホールコンピュータ**: 各台の打ち込み玉数、払い出し玉数、当たり、ハズレ、大当たり回数等を管理するコンピュータ
- **呼び出しランプ**: 台の上部に付いているランプ。ボタンを押すと店員が来る
- **トランス**: 電圧を変える変圧器（100V→24V）

### 遊技機の規格と歴史（CR機 → P機 → e機 → LT）

| 項目 | CR機 | P機 | e機（スマパチ） |
|------|------|------|------|
| 時期 | 1990年代～2019年 | 2018年2月～ | 2023年～ |
| 方式 | プリペイドカード式 | プリペイドカード式 | 封入式（電子管理） |
| 大当たり下限確率 | なし | 1/320 | 1/350 |
| 最大ラウンド数 | 16R | 10R | 10R |
| 最大出玉 | 約2,400個 | 約1,500個 | 約9,600個（LT対応時） |
| 設定機能 | なし | 6段階可 | 6段階可 |
| 特記 | 確変搭載、高い連チャン性能 | 遊タイム導入、射幸性抑制 | Cタイム・コンプリート機能搭載 |

- **CR機**: Card Reader機の略。プリペイドカード方式で現金を直接使用しない。2019年に新規導入終了
- **P機**: 2018年の風営法改正に伴い登場。最大ラウンド数がCR機の16Rから10Rに削減。6段階設定搭載が可能に
- **e機（スマパチ）**: 2023年登場の封入式遊技機。物理的な玉を使用せず電子管理。玉詰まりがない
- **遊タイム**: P機で導入された機能。一定回転数ハマると自動的に時短が発動するシステム
- **Cタイム**: e機で搭載可能になった新機能
- **コンプリート機能**: e機に標準搭載。出玉上限に到達した時に強制終了する機能
- **LT（ラッキートリガー）**: 2024年3月導入。特定条件で一時的に高出玉性能を発揮する機能。最大出玉約9,600個（従来の6,400個から緩和）。ライトミドル～甘デジ設計が対象

### パチンコ機種の基本概念
- **確率**: デジタル1回転ごとに抽選されている大当たりの確率
- **確変（確率変動）**: 大当たり後、大当たり確率がアップする機能
- **時短**: デジタル変動の短縮＋電チュー開放による回転効率アップ機能
- **ST**: 確変回数切り。規定回転数までは確変状態をキープする
- **回転率**: 1000円あたりの回転数。ボーダーラインと比較するための重要指標
- **ボーダーライン**: 1000円あたり何回転すれば収支±0になるかの数値
- **保留**: スタートチャッカーに入った玉数を台が記憶（最高4つ）
- **ラウンド**: 大当たり時にアタッカーが開閉する1回の動作（CR機:最大16R、P機/e機:最大10R）
- **甘デジ（遊パチ）**: おおむね1/100前後の低確率機種。手軽に安く遊べる

### パチスロの基本概念
- **設定**: 出玉率を決める段階（設定1～6）
- **合算確率**: ビッグボーナスとレギュラーボーナスの当選確率を合わせたもの
- **小役**: ボーナス以外の役。各小役で払い出し枚数が異なる

### ホール運営用語
- **稼動率**: 客が打って機械が稼動している割合
- **釘調整（ゲージメンテナンス）**: ハンマー等で釘や風車の角度を調整し、玉の動きや出玉率を調整すること
- **貯玉**: 獲得した玉を貯めておくこと
- **再プレイ**: 貯玉を引き出してプレイすること
- **打ち止め**: 定量数に達した時に遊戯をストップさせること
- **ラムクリ**: 確変・時短状態のリセット・クリア
- **みなし機**: 検定・認定の切れている台

## データの定義

### ホールコンピュータ基本項目

| 用語 | 意味 |
|------|------|
| **TS** | 特賞確率（初当り確率）。ホールコンピュータで算出される実績値 |
| **TY** | 大当りで獲得できる玉数やコイン枚数 |
| **TO** | 大当り1回あたりの所要時間（分）。出玉速度の計算に使用 |
| **BA値** | アウト玉100個（約2分）に対して何個返球（セーフ玉）したかを表す数値。基本性能のデータ化 |
| **アウト玉** | 打ち出された玉数（消費玉） |
| **セーフ玉** | お客様が獲得した玉（払い出し玉） |
| **差玉** | アウト玉とセーフ玉の差 |
| **出玉率** | 打った玉数に対する払い出された玉数の割合 |
| **出玉速度** | TY÷TO×100。1分間あたりの出玉数 |
| **稼動率** | 客が打って機械が稼動している割合 |
| **回転率** | 1000円あたりのデジタル回転数 |
| **スランプグラフ** | 大当たりやハマリの波を折れ線グラフで表したもの |
| **型式名** | 保通協に申請した時点での機種名 |
| **スペック（spec）** | specification の略。機種の仕様・性能 |

### P-Brain シェア分析の項目

出典: P-Brainマニュアル「シェア分析」

シェア分析は、遊技種別・機種タイプ別・機種別に売上・粗利・設置シェアの相互関係と推移を検証し、仕掛け・オペレーションに活かすための機能。

#### 検索条件
- **期間**: デフォルトは当月。任意選択可能
- **エリア・店舗**: 対象のエリアまたは店舗を選択
- **種別**: 遊技種別（パチンコ/スロット等）
- **機種カテゴリ**: 遊技種別 → 機種タイプ別 → 機種別の階層で選択可能
- **タグ**: 機種タイプでは分類されていない詳細条件（例: e機+ハイミドル）を任意指定
- **モード**: メーカーグループ別等の切り替えが可能

#### 表示モード（6種類）

| 表示モード | 説明 |
|------------|------|
| **打込** | 台あたりの打込玉数（アウト玉） |
| **台売上** | 台あたりの売上金額 |
| **台粗利** | 台あたりの粗利金額 |
| **台数** | 設置台数 |
| **シェア** | 各項目についての総数に占める割合（%） |
| **支持率** | 設置台シェアに対する各指標シェアの比率。台数に見合った成果を出しているかを評価する指標 |

#### 支持率の定義と計算式

| 支持率 | 計算式 | 意味 |
|--------|--------|------|
| **打込支持率** | 打込シェア ÷ 台数シェア × 100 | 設置台占有率の中でどれだけ打込シェアを有しているか |
| **台売上支持率** | 台売上シェア ÷ 台数シェア × 100 | 設置台占有率の中でどれだけ台売上シェアを有しているか |
| **台粗利支持率** | 台粗利シェア ÷ 台数シェア × 100 | 設置台占有率の中でどれだけ台粗利シェアを有しているか |

例: 打込シェア10%・台数シェア10% → 支持率100%

#### 支持率の色別評価基準

| 色 | 支持率 | 評価 |
|----|--------|------|
| 青 | 130%以上 | 台数以上に稼いでいる（優良） |
| 黄 | 100～129% | 台数相応の成果 |
| 赤 | 70～99% | 台数に対してやや不足 |
| 無色 | 69%以下 | 台数に対して大幅に不足 |

#### 台数シェア別の分類

| 区分 | 台数シェア | 位置づけ |
|------|-----------|---------|
| ① | 4%以上 | 主力機種 |
| ② | 3～4%未満 | 準主力 |
| ③ | 2～3%未満 | 中堅 |
| ④ | 1～2%未満 | 少数 |
| ⑤ | 1%未満 | 最小 |

#### 運用の考え方
- **打込支持率70%以下**の機種 → 無駄出しの改善が必要
- **主力機種（台数シェア4%以上）** → 主力としての動きをしているか、利益配分は適正かを確認
- 支持率の帳票から、増減台・低貸しへの移設など運用方法の判断を行う
- **メーカーグループ別**: 販売製造メーカーは異なるが大本が同一のメーカーをカテゴライズして分析（例: サミー＝サミー/タイヨーEC/ロデオ、平和＝平和/オリンピア/アムテックス）

#### P-Brainにおける機種評価の基準値

P-Brainでは「導入4週目」の「打込」および「打貢献率（打込支持率）」を機種の良し悪しの判断基準としている。

**パチンコの優良機種基準:**
- 導入4週目の打込: **20,000以上**をキープしているか
- 導入4週目の打貢献率: **220%以上**
- 導入8週目の打貢献率: **140%以上**
- 上記を満たす機種は中長期で貢献する「固定機」候補

**パチスロの優良機種基準:**
- 導入4週目の打込: **10,000以上**をキープしているか
- 導入4週目の打貢献率: **170%以上**
- 導入8週目の打貢献率: **130%以上**（ジャガー除く）

**打貢献率の活用ポイント:**
- 打込だけでなく打貢献率を見ることで、繁忙期・閑散期に関わらず同様の比較が可能
- 打貢献率130%超がP-Brainでの一つの目安
- 機種選定（中古購入・増台時）の参考指標として活用
